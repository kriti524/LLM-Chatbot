# LLM-Chatbot

LLM Chatbot Web App | Python, LangChain, Streamlit, OpenAI / Llama2

Built an interactive chatbot using Streamlit and LangChain, with OpenAI API integration (via API key) for cloud-based responses and LLaMA2 (via Ollama) as the default local LLM for offline usage. Used ChatPromptTemplate for prompt structuring and StrOutputParser for handling model outputs, enabling seamless switching between local and OpenAI-powered responses.
